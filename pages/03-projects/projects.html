Freelance company Luohaus: WordPress website project

Personal project
(10/2024 - 01/2025)
https://www.luohaus.fi/

Tools used:

WordPress, various WordPress plugins, XAMPP, MySQL, Google Translation API, VS Code, HTML, CSS

Summary:

The web service project was implemented using the WordPress content management system. The purpose of the project was to create a portfolio website for a small company offering graphic design services, so the customers could view the company's work and easily contact the entrepreneur.

The website design focused on the accessibility, usability, necessary functionalities and on a pleasant graphic appearance of the website. In addition to the version that was published on the client's web hosting service, an online store version was also created in accordance with the professional requirements for Taitotalo's skills demonstration.

Once a suitable WordPress theme had been selected, the theme was modified to suit the purpose, for example with different fonts, colors and structures. Various WordPress plugins corresponding to the required functionalities were selected, such as Duplicator, AIOSEO, WooCommerce, etc. Content provided by the client was added to the pages according to the graphic instructions, and the appearance of the site was corrected if necessary. The site was published on a Finnish web hosting service called Hostingpalvelu.

The project was successful according to the client, and it is planned to be developed further later.

//Tuodaan tarvittavat Reactin osat
import { StrictMode } from "react";
import { createRoot } from "react-dom/client";
import { useState } from "react";
import './index.css';
import App from './App';

//Komponentti 'Square' saa ruudukon arvon 'Board' komponentilta
//Luodaan 'onSquareClick', joka klikkauksella suorittaa funktion 'handleClick'
// -> joka taas luo arvon ruudukkoon funktion 'Board' avulla
function Square({ value, onSquareClick }) {
    return (
    <button className="square" onClick={onSquareClick}>
        {value}
    </button>
    );
}

//Luodaan ruudukko ja komponentti 'Board' ruudukon ja sen arvojen hallinnointiin
export default function Board() {
    const [squares, setSquares] = useState(Array(9).fill(null));
//komponentti 'handleClick' luo ruudukkoon kirjaimen 'X'
    function handleClick(i) {
        const nextSquares = squares.slice();
        nextSquares[i] = "X";
        setSquares(nextSquares);
    }
//Luodaan ruudukko ja määritetään mitä tapahtuu kun ruutua klikkaa
//Funktio 'Board' hallitsee ruudukon tilaa
    return (
        <>
            <div className="board-row">
                <Square value={squares[0]} onSquareClick={() => handleClick(0)} />
                <Square value={squares[1]} onSquareClick={() => handleClick(1)} />
                <Square value={squares[2]} onSquareClick={() => handleClick(2)} />
            </div>

            <div className="board-row">
                <Square value={squares[3]} onSquareClick={() => handleClick(3)} />
                <Square value={squares[4]} onSquareClick={() => handleClick(4)} />
                <Square value={squares[5]} onSquareClick={() => handleClick(5)} />
            </div>

            <div className="board-row">
                <Square value={squares[6]} onSquareClick={() => handleClick(6)} />
                <Square value={squares[7]} onSquareClick={() => handleClick(7)} />
                <Square value={squares[8]} onSquareClick={() => handleClick(8)} />
            </div>
        </>
    );
}
